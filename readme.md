    The primary main page of this website should not require and special resource or compilation.

However, some of the projects contained within, like the RoboSnake and TS Dungeon will require
NPM webpack to build correctly. If they are copied directly from the folders contained within,
they should be pre-built and able to run independently of this website. The python scripts do
require pytorch and the fashion MNIST data set. However, I am pretty sure that they will download
their own required resources when run.
